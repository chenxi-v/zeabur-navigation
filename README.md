# my-zeabur-app
1. 个人网址导航页
2. 复刻仓库后Zeabur直接选GitHub导入即可
```
zeabur-navigation/
├── index.html             # 主页面（包含css和天气插件）
├── nav-data.json          # 导航数据配置
├── zeabur.json            # Zeabur平台配置
├── assets/
│   ├── backgrounds/       # 背景图片目录
│   └── icons/             # 网站图标目录
└── README.md              # 部署说明
```
